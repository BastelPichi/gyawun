## Features
- [x] Play (almost) any song or video from YouTube Music
- [x] Background playback
- [x] Search for songs and playlists
- [x] Reorder songs in playlist or queue
- [x] Dark Mode
- [x] Multiple Language Support
- [ ] Add to favourites
- [ ] Save history locally
- [ ] Save playlist
